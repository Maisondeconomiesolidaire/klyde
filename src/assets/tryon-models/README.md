# Mannequins pour l'essayage virtuel (FASHN Try-On Max)

Déposez ici les images des mannequins (une image = un modèle sélectionnable
lors de la génération d'essayage). Formats acceptés : `.jpg`, `.jpeg`, `.png`,
`.webp`.

Le **nom du fichier** (sans extension) devient le **nom affiché** du modèle
dans le sélecteur — ex. `Femme brune.jpg` → « Femme brune ». Les tirets et
underscores sont remplacés par des espaces.

Les modèles sont détectés automatiquement au build (aucun code à modifier) :
il suffit d'ajouter/retirer un fichier dans ce dossier.
